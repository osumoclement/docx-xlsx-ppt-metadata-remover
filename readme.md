# Office Document Metadata Remover

# How it works

This is an addon that strips all metadata from Office documents like Word, Excel and PowerPoint.

Supported formats: docx, xlsx, pptx

Tested on ArchLinux, should work on other platforms as long as you have dotnet-sdk
# Installation
Install dotnet-sdk

Git clone this project & cd into

dotnet run -- "/path/to/your/document"

That's it, document should be stripped of metadata

Feel free to ask any questions, submit feature/bug reports
